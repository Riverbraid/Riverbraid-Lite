### 1. Riverbraid-Core: Foundational Logic
**Foundational logic and invariant definitions for the Riverbraid cluster.**

#### **Overview**
`Riverbraid-Core` is the substrate for a **governance-locked cognitive architecture**. It separates capacity control from intelligence, providing a stable, auditable environment. Unlike traditional AI frameworks, Riverbraid is **deterministic over adaptive**; its control logic does not learn, drift, or personalize.

#### **Core Functions**
* **Capacity Control Primitives:** Decides *how much* a system should engage.
* **Invariant Enforcement:** Defines the "hard" rules that cannot be violated.
* **Identity Contracts:** Manages `identity.contract.json` to reconcile governed files.
