# Security Policy

## Supported Versions
We only support the latest tagged releases of the Gold Cluster (currently v1.1.0) and Riverbraid-Core (v0.1.3). 

## Reporting a Vulnerability
Safety and Refusal integrity are paramount. If you discover a logic bypass or a state-drift vulnerability, please do not use public issues.

1. Email the maintainer directly or use the "Report a vulnerability" button on the specific repository.
2. Provide a deterministic reproduction case.
3. Allow 48 hours for an "Honest Advisor" assessment before public disclosure.
